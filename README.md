# Overview CSE 310 - Web-Based To-Do List

# Author : Diogo Rangel Dos Santos

## Overview
This application is a simple, browser-based Task Manager. It allows users to input tasks, mark them as finished by clicking the text, and remove them entirely. It was built to demonstrate proficiency in JavaScript DOM manipulation.
